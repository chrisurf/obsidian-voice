Piper Neural TTS — Feasibility Harness

Runs Piper (VITS ONNX + espeak-ng phonemizer) fully on-device in the browser via @mintplex-labs/piper-tts-web — no server, no API key. This validates the one open question for Piper: does in-browser phonemization work, and what are the size / speed / quality numbers per language? First run of a voice downloads it from the Hugging Face CDN into the Origin Private File System (OPFS); afterwards it is served offline from there.
